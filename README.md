# 📚 Library Management System

A simple and efficient **Library Management System** developed as a DBMS mini project.  
This system helps in managing library operations like storing book records, issuing books, and tracking returns.

## 🚀 Features

- 📖 Add and manage book records  
- 👩‍🎓 Manage student details  
- 🔄 Issue and return books  
- 🔍 Search books easily  
- 📊 Maintain proper database records  

## 🛠️ Technologies Used

- 💾 DBMS (Database Management System)  
- 🗄️ SQL  
- 💻 html ,CSS, javascript
